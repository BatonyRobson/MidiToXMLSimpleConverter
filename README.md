# Конвертер MIDI файлов в XML

Миди файлы, находящиеся в папке с .py файлом, при запуске этого файла конвертируются в XML файл, который можно закинуть в Премьер, чтобы получить маркеры в началах нот. Полезно для монтажа под свою музыку.

Проверял на простой мелодии без полифонии. Закидывал в Премьер 2024.3 и работает. Можете попробовать в другие видео редакторы позакидывать, авось сработает.

# Как пользоваться?
1) Поставить в систему питон 3.10+
2) Поставить миди библиотеку mido - устанавливается командой в консоли (win + R, cmd):

pip install mido

3) Положить в одну папку MidiToXML.py и ваши .mid файлы
4) запустить скрипт
5) ввести бпм, с которым писали трек
6) ввести ФПС, с которым вы собираетесь делать секвенцию в Премьере
7) Закрыть скрипт
8) Перетянуть получившийся xml файл в премьер. У вас появится секвенция с маркерами
9) В меню Маркеры включить копипасту маркеров снизу
10) Взять что угодно в качестве клипа и засунуть его в новую секвенцию, чтобы хватило по длине на все маркеры
11) Поставить ин аут маркеры (возможно не нужен этот шаг)
12) Скопировать свой клип в ту секвекнцию, где вы собираетесь работать. Вместе с клипом перенесутся маркеры секвенции.
13) Удалить этот ненужный более клип и наслаждаться маркерами.
