# Конвертер MIDI файлов в XML

Миди файлы, находящиеся в папке с .py файлом, при запуске этого файла конвертируются в XML файл, который можно закинуть в Премьер, чтобы получить маркеры в началах нот. Полезно для монтажа под свою музыку.

Проверял на простой мелодии без полифонии. Закидывал в Премьер 2024.3 и работает. Можете попробовать в другие видео редакторы позакидывать, авось сработает.

# Как пользоваться?
1) Поставить в систему питон 3.10+
2) Поставить миди библиотеку mido - устанавливается командой в консоли (win + R, cmd):

pip install mido

3) Скачать отсюда файл MidiToXML.py
4) Положить его в одну папку с вашими .mid файлами
5) запустить скрипт
6) ввести бпм, с которым писали трек (если динамичный БПМ у вас, то сори)
7) ввести ФПС, с которым вы собираетесь делать секвенцию в Премьере
8) Закрыть скрипт после выполнения
9) Перетянуть получившийся xml файл в премьер. У вас появится секвенция с маркерами

Если нужно скопировать маркеры в свою секвенцию:

11) В меню Маркеры включить копипасту маркеров снизу
<img width="961" height="740" alt="image" src="https://github.com/user-attachments/assets/684c1add-9b84-4322-b195-1179b434ac86" />

13) Взять что угодно в качестве клипа и засунуть его в новую секвенцию, чтобы хватило по длине на все маркеры
14) Поставить ин аут маркеры (возможно не нужен этот шаг)
15) Скопировать свой клип в ту секвекнцию, где вы собираетесь работать. Вместе с клипом перенесутся маркеры секвенции.
16) Удалить этот ненужный более клип и наслаждаться маркерами.
